# Amazon Clone (Landing Page)

This project is a static Amazon-style landing page created using HTML and CSS. It recreates the basic layout and visual structure of an e-commerce homepage for frontend practice.

## Features

- Navbar with search layout
- Product and category sections
- Banner area
- Footer layout
- Responsive styling

## Technologies Used

- HTML
- CSS

## How to Run

Open `index.html` in a browser.
